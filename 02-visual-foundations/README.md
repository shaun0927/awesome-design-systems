## Key Takeaways

- **Color in three layers**: Structure palettes as neutral (gray scale), brand (primary actions), and semantic (success/error/warning) to cover 95% of UI needs
- **Typography is 80% of UI**: Master size, weight, color, and line-height to create visual hierarchyâ€”this single skill transforms interface clarity more than any other
- **Rem-based spacing for scalability**: Use rem units (not pixels) with geometric progressions (e.g., 0.25rem â†’ 0.5rem â†’ 1rem â†’ 2rem) to ensure consistent grouping and separation
- **Component height consistency**: Align all interactive components (buttons, inputs, selects) to the same height scale (S/M/L) by harmonizing typography + padding + border
- **Depth creates interest**: Layer shades of the same color (lightness adjustments in HSL) and apply shadows (small/medium/large) to transform flat UIs into engaging experiences
