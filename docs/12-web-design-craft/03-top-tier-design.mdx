## Less is More

핵심 기능에 집중하고 불필요한 요소를 제거합니다.

```tsx
// ❌ 나쁜 예: 복잡한 레이아웃
<Hero>
  <Particles />
  <AnimatedBackground />
  <GlowEffect />
  <Title />
  <Subtitle />
  <Description />
  <CTA primary />
  <CTA secondary />
  <CTA tertiary />
</Hero>

// ✅ 좋은 예: 핵심만 남김
<Hero>
  <Title>Transform Your Workflow</Title>
  <input placeholder="Enter your email" />
  <button>Get Started</button>
</Hero>
```

## 간격 시스템 (4의 배수 기반)

```css
:root {
  /* Spacing scale */
  --space-1: 0.25rem;  /* 4px */
  --space-2: 0.5rem;   /* 8px */
  --space-3: 0.75rem;  /* 12px */
  --space-4: 1rem;     /* 16px */
  --space-5: 1.25rem;  /* 20px */
  --space-6: 1.5rem;   /* 24px */
  --space-8: 2rem;     /* 32px */
  --space-10: 2.5rem;  /* 40px */
  --space-12: 3rem;    /* 48px */
}
```

## 타이포그래피 시스템

```css
:root {
  /* Type scale */
  --font-size-xs: 0.75rem;    /* 12px */
  --font-size-sm: 0.875rem;   /* 14px */
  --font-size-base: 1rem;     /* 16px */
  --font-size-lg: 1.125rem;   /* 18px */
  --font-size-xl: 1.25rem;    /* 20px */
  --font-size-2xl: 1.5rem;    /* 24px */
  --font-size-3xl: 1.875rem;  /* 30px */
  --font-size-4xl: 2.25rem;   /* 36px */
  --font-size-5xl: 3rem;      /* 48px */

  /* Line heights */
  --leading-tight: 1.25;
  --leading-normal: 1.5;
  --leading-relaxed: 1.75;
}
```

### Line Height 규칙

```css
/* 큰 제목: 낮은 line-height */
h1 {
  font-size: var(--font-size-5xl);
  line-height: var(--leading-tight); /* 1.25 */
}

/* 본문: 중간 line-height */
p {
  font-size: var(--font-size-base);
  line-height: var(--leading-normal); /* 1.5 */
}

/* 작은 텍스트: 높은 line-height */
.small-text {
  font-size: var(--font-size-sm);
  line-height: var(--leading-relaxed); /* 1.75 */
}
```

## 색상 시스템 (최소 4색)

```css
:root {
  /* Core colors */
  --color-dark: hsl(220, 15%, 20%);        /* 텍스트 */
  --color-light: hsl(220, 15%, 98%);       /* 배경 */
  --color-accent-1: hsl(210, 100%, 50%);   /* Primary CTA */
  --color-accent-2: hsl(270, 100%, 50%);   /* Secondary */

  /* Semantic colors */
  --color-text: var(--color-dark);
  --color-bg: var(--color-light);
  --color-primary: var(--color-accent-1);
  --color-primary-hover: hsl(210, 100%, 40%);
}

@media (prefers-color-scheme: dark) {
  :root {
    --color-text: hsl(220, 15%, 95%);
    --color-bg: hsl(220, 15%, 10%);
    --color-primary: hsl(210, 100%, 60%);
  }
}
```

### Tailwind 등가물

```tsx

  color: 'white',
  padding: 'var(--space-4) var(--space-6)'
}}>

// Tailwind
<button className="bg-blue-500 text-white px-6 py-4">
```

## 컴포넌트 시스템

```tsx
// Button variants
interface ButtonProps {
  variant?: 'primary' | 'secondary';
  children: React.ReactNode;
}

export function Button({ variant = 'primary', children }: ButtonProps) {
  return (
    <button className={`btn btn-${variant}`}>
      {children}
    </button>
  );
}
```

```css
.btn {
  padding: var(--space-3) var(--space-6);
  border-radius: 6px;
  font-weight: 600;
  transition: all 0.2s;
}

.btn-primary {
  background: var(--color-primary);
  color: white;
}

.btn-primary:hover {
  background: var(--color-primary-hover);
  transform: translateY(-2px);
}

.btn-secondary {
  background: transparent;
  color: var(--color-primary);
  border: 2px solid var(--color-primary);
}
```

## 계층 구조 (Hierarchy)

### 강조 순서: 대비 → 굵기 → 크기

```css
/* 1단계: 대비로 약화 */
.secondary-text {
  color: hsl(220, 15%, 60%); /* 낮은 대비 */
}

/* 2단계: 굵기 증가 */
.primary-text {
  font-weight: 700;
}

/* 3단계: 크기 증가 */
h1 {
  font-size: var(--font-size-5xl);
  font-weight: 700;
  color: var(--color-text);
}

h2 {
  font-size: var(--font-size-3xl);
  font-weight: 600;
  color: var(--color-text);
}

p {
  font-size: var(--font-size-base);
  font-weight: 400;
  color: hsl(220, 15%, 50%); /* 낮은 대비 */
}
```

### 실전 예시

```tsx
<article>
  <h1 className="text-5xl font-bold text-gray-900">
    Main Headline
  </h1>
  <p className="text-base text-gray-500 mt-2">
    Supporting information in muted color
  </p>
  <button className="mt-4 bg-blue-500 text-white font-semibold">
    Call to Action
  </button>
</article>
```
