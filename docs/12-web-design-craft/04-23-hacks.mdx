## 색상 시스템

### 1. HSL 색상 체계

```css
/* HSL 구조: Hue(0-360), Saturation(0-100%), Lightness(0-100%) */
:root {
  --primary-hue: 210;
  --color-primary: hsl(var(--primary-hue), 80%, 50%);
  --color-primary-light: hsl(var(--primary-hue), 80%, 70%);
  --color-primary-dark: hsl(var(--primary-hue), 80%, 30%);
}

.button {
  background: var(--color-primary);
}

.button:hover {
  background: var(--color-primary-dark); /* L값만 변경 */
}
```

### 2. 자동 색상 팔레트 생성

```css
:root {
  --hue-primary: 210;

  /* Primary/Secondary - 채도/명도 조정 */
  --color-primary: hsl(var(--hue-primary), 80%, 50%);
  --color-secondary: hsl(var(--hue-primary), 60%, 70%);

  /* Tertiary/Accent - Hue를 ±60도 이동 */
  --color-tertiary: hsl(calc(var(--hue-primary) - 60), 70%, 55%);
  --color-accent: hsl(calc(var(--hue-primary) + 60), 70%, 55%);
}
```

### 5. 다크모드 원라인 구현

```css
:root {
  color-scheme: light dark;
}

@media (prefers-color-scheme: dark) {
  :root {
    --color-bg: hsl(210, 20%, 10%);
    --color-text: hsl(210, 10%, 90%);
    --color-accent: hsl(210, 80%, 65%);
  }
}
```

## HTML 네이티브 기능

### 7. contenteditable (인라인 편집)

```html
<div contenteditable="true" class="editable-note">
  <h3>Q: 이 서비스는 어떻게 사용하나요?</h3>
  <p>여기에 답변을 입력하세요...</p>
</div>
```

```css
[contenteditable="true"] {
  border: 2px dashed transparent;
  padding: 1rem;
  border-radius: 4px;
}

[contenteditable="true"]:focus {
  border-color: var(--color-primary);
  outline: none;
}
```

### 8. inert (비활성화)

```tsx
// 모달이 열릴 때 배경 비활성화
<main inert={isModalOpen}>
  <button>클릭 안 됨</button>
  <a href="/">링크 작동 안 함</a>
</main>
<dialog open={isModalOpen}>
  <h2>모달 내용</h2>
</dialog>
```

### 10. dialog 태그 (네이티브 모달)

```tsx
export function Modal({ children }: { children: React.ReactNode }) {
  const dialogRef = useRef<HTMLDialogElement>(null);

  return (
    <>
      <button onClick={() => dialogRef.current?.showModal()}>
        Open Modal
      </button>
      <dialog ref={dialogRef}>
        {children}
        <button onClick={() => dialogRef.current?.close()}>
          Close
        </button>
      </dialog>
    </>
  );
}
```

```css
dialog {
  border: none;
  border-radius: 8px;
  padding: 2rem;
  box-shadow: 0 10px 25px rgba(0, 0, 0, 0.3);
}

dialog::backdrop {
  background: rgba(0, 0, 0, 0.5);
}
```

### 15. details/summary (네이티브 Accordion)

```html
<details class="accordion">
  <summary>펼치기 / 접기</summary>
  <p>숨겨진 내용...</p>
</details>
```

```css
summary {
  cursor: pointer;
  font-weight: 600;
  padding: 1rem;
  background: var(--color-bg);
  border-radius: 4px;
}

details[open] summary {
  border-bottom: 1px solid #e0e0e0;
}
```

```tsx
// 진행률 표시

  value={0.85}
  min={0}
  max={1}
  low={0.3}
  high={0.7}
  optimum={0.9}
>
  85%
</meter>
```
